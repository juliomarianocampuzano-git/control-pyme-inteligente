## 📌 Descripción

**Control PyME Inteligente** es un sistema de gestión empresarial completo que corre directamente en tu navegador — sin servidor, sin base de datos externa, sin conexión a internet.

Diseñado para dueños de PyMEs que necesitan controlar su negocio de forma simple, rápida y privada.

---

## ✨ Funcionalidades

| Módulo | Descripción |
|--------|-------------|
| 🧾 **Facturación** | Facturas, cotizaciones y pedidos |
| 📦 **Inventario** | Control de stock con alertas |
| 💰 **Finanzas** | Ingresos, egresos, caja y balance |
| 👥 **Clientes** | CRM con historial y segmentación |
| 🤖 **IA Local** | Análisis y sugerencias offline |
| 📊 **Reportes** | Gráficos y exportación Excel |
| 🔒 **Seguridad** | Contraseña con hash SHA-256 |
| 💾 **Backup** | Exportación e importación completa |

---

## 🚀 Cómo usar

1. **Abrí el link** → [Control PyME Inteligente](https://juliomarianocampuzano-git.github.io/control-pyme-inteligente/)
2. **Cargá tus datos** con las plantillas CSV incluidas
3. **Visualizá reportes** y exportá en Excel

No requiere instalación. Funciona 100% offline desde el navegador.

---

## 🔒 Privacidad y seguridad

- ✅ **100% offline** — ningún dato sale de tu computadora
- ✅ **Sin telemetría** — no hay tracking ni analytics
- ✅ **Datos en tu PC** — almacenados en IndexedDB del navegador
- ✅ **Sin suscripción** — pago único, tuyo para siempre
