# 🧠 Control PyME Inteligente v1.1

> **Dashboard de gestión offline para pequeñas y medianas empresas (PyMEs)**
> > Aplicación HTML/JS de archivo único — sin servidor, sin suscripción, sin internet.
> >
> > ![Version](https://img.shields.io/badge/versión-1.1-blue) ![License](https://img.shields.io/badge/licencia-MIT-green) ![Platform](https://img.shields.io/badge/plataforma-Windows-lightgrey) ![Offline](https://img.shields.io/badge/modo-offline-orange)
> >
> > ---
> >
> > ## 📌 Descripción
> >
> > **Control PyME Inteligente** es un sistema de gestión empresarial completo desarrollado como una sola aplicación HTML/JavaScript que corre directamente en el navegador, sin necesidad de servidor, base de datos externa ni conexión a internet.
> >
> > Diseñado para dueños de PyMEs que necesitan controlar su negocio de forma simple, rápida y privada.
> >
> > ---
> >
> > ## ✨ Funcionalidades principales
> >
> > | Módulo | Descripción |
> > |---|---|
> > | 🧾 **Facturación** | Creación y seguimiento de facturas, cotizaciones y pedidos |
> > | 📦 **Inventario** | Control de stock con alertas de stock mínimo |
> > | 💰 **Finanzas** | Ingresos, egresos, caja y balance |
> > | 👥 **Clientes** | CRM con historial, contactos y segmentación |
> > | 🤖 **IA Local** | Motor de inteligencia artificial offline para análisis y sugerencias |
> > | 🔒 **Seguridad** | Sistema de contraseña con hash SHA-256 para proteger datos |
> > | 📊 **Reportes** | Gráficos y exportación de datos |
> > | 🏢 **Multi-empresa** | Gestión de múltiples empresas con datos separados |
> > | 📧 **Comunicaciones** | Historial de comunicaciones y seguimiento de clientes |
> > | 💾 **Backup** | Exportación e importación de datos completa |
> >
> > ---
> >
> > ## 🛠️ Stack tecnológico
> >
> > - **Frontend:** HTML5, CSS3, JavaScript (ES6+) — single-file SPA
> > - - **Almacenamiento:** IndexedDB (base de datos local del navegador)
> >   - - **Seguridad:** Web Crypto API — hashing SHA-256 con salt
> >     - - **IA:** Motor de inferencia local sin dependencias externas
> >       - - **Instalador:** Inno Setup 6 — instalador Windows (.exe)
> >         - - **Compatibilidad:** Cualquier navegador moderno (Chrome, Edge, Firefox)
> >          
> >           - ---
> >
> > ## 📁 Estructura del proyecto
> >
> > ```
> > control-pyme-inteligente/
> > ├── 1_APP_PORTABLE/          # Aplicación principal (dashboard.html)
> > ├── 2_MANUAL_USUARIO/        # Manual de usuario en PDF
> > ├── 3_PLANTILLAS_NEGOCIO/    # Plantillas Excel para el negocio
> > ├── 4_PLANTILLAS_OPERATIVAS/ # Plantillas Excel operativas
> > ├── 5_BACKUPS_DEL_CLIENTE/   # Carpeta de backups exportados
> > ├── 6_DOCUMENTOS_COMERCIALES/# Presentación, condiciones de uso
> > └── 7_INSTALADOR_WINDOWS/    # Instalador .exe compilado con Inno Setup
> > ```
> >
> > ---
> >
> > ## 🚀 Instalación
> >
> > ### Opción 1 — Instalador Windows (recomendado)
> > 1. Descargá `PyME_Inteligente_Setup_v1.1_FINAL.exe` desde la carpeta `7_INSTALADOR_WINDOWS/`
> > 2. 2. Ejecutá el instalador
> >    3. 3. Abrí el acceso directo en tu escritorio
> >      
> >       4. ### Opción 2 — Modo portable
> >       5. 1. Descargá el repositorio
> >          2. 2. Abrí `1_APP_PORTABLE/dashboard.html` directamente en tu navegador
> >             3. 3. ¡Listo! No requiere instalación
> >               
> >                4. ---
> >               
> >                5. ## 🔒 Privacidad y seguridad
> >               
> >                6. - **100% offline** — ningún dato sale de tu computadora
> > - **Sin telemetría** — no hay tracking ni analytics
> > - - **Contraseña opcional** — protección con hash SHA-256 + salt
> >   - - **Datos en tu PC** — almacenados en IndexedDB del navegador
